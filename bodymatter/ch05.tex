\chapter{模板定制与注意事项}

\section{切换论文类型}

在主文件中修改 \type{type} 和 \type{degree} 参数即可切换论文类型。

博士学术学位论文：

\starttyping
\setvariables [thesis] [
  type=doctor,
  degree=academic,
  ...
]
\stoptyping

硕士专业学位论文：

\starttyping
\setvariables [thesis] [
  type=master,
  degree=professional,
  ...
]
\stoptyping

本科毕业论文（无学位类型区分）：

\starttyping
\setvariables [thesis] [
  type=bachelor,
  ...
]
\stoptyping

\section{涉密论文设置}

涉密论文需在封面右上角标注密级和保密年限。
设置 \type{secret-level} 和 \type{secret-year} 参数即可：

\starttyping
\setvariables [thesis] [
  secret-level={秘密},
  secret-year={五年},
  ...
]
\stoptyping

\section{字体配置}

模板默认使用 Source Han Serif CN（思源宋体）
和 Source Han Sans CN（思源黑体）。
如果系统未安装这些字体，
可在 \type{fduthesis_env.tex} 中替换为其他字体：

\starttyping
% 替换为 Noto SC 字体
\definefontfamily [mainface] [rm] [Noto Serif SC]
\definefontfamily [mainface] [ss] [Noto Sans SC]
\stoptyping

替换后需执行 \type{mtxrun --script fonts --reload} 刷新字体数据库。

\section{添加章节}

正文各章节分别存放在 \type{bodymatter/} 目录下的独立文件中。
添加新章节时，创建新文件
（如 \type{bodymatter/ch06.tex}）并在主文件中加入：

\starttyping
\input bodymatter/ch06
\stoptyping

\section{注意事项}

\startitemize[n]
  \item 论文封面格式须严格按照规范标准样本制作，
        不得随意修改封面宏的布局。
  \item 中文论文题目不超过 30 个汉字。
  \item 参考文献格式应符合 GB/T 7714—2015 国家标准。
  \item 硕士论文采用篇末注，博士论文采用章末注；
        也可适当采用脚注。
  \item 论文须双面打印，用线装或热胶装订。
  \item 文科硕士论文一般不少于 3 万字，博士论文不少于 10 万字。
  \item 独创性声明和授权声明须打印后手写签名。
\stopitemize

\stopcomponent
