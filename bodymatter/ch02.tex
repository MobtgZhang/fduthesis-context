\chapter{文本排版}

\section{字体与字号}

根据论文规范，正文使用宋体小四号（12pt），标准字符间距，行距 20 磅。
模板已自动配置好这些参数，用户无需手动设置。

章标题使用三号（16pt）黑体加粗居中，节标题使用四号（14pt）黑体加粗，
小节标题使用小四号（12pt）黑体加粗。

模板提供了中文字号命令，可在需要时手动切换字号：

\starttyping
\chuhao     % 初号 42pt
\xiaochuhao % 小初 36pt
\yihao      % 一号 26pt
\erhao      % 二号 22pt
\xiaoerhao  % 小二 18pt
\sanhao     % 三号 16pt
\sihao      % 四号 14pt
\xiaosihao  % 小四 12pt（正文默认）
\wuhao      % 五号 10.5pt
\stoptyping

正文段落自动首行缩进 2 个汉字宽度。

基于神经网络的方法在机器翻译、文本分类、
情感分析、问答系统等任务上取得了显著突破。
特别是 Transformer 架构的提出，
彻底改变了自然语言处理的研究范式。
